Описание:

Содержание:
AccessToFile.bat - главный файл скрипта.
Папка "extractor" содержит основные файлы для работы скрипта.

Описание AccessToFile.bat:
В adb backup следует указать полное название пакета приложения. Пример: knyaseff.net.lab2_writefile
После выполнения adb backup создается файл backup.ab в папке extractor.
Формат преобразования backup в обычный .tar архив:
java -jar abe.jar unpack [имя файла backup] [имя нового архива] [пароль, введенный при создании backup]
Пример: 
java -jar abe.jar unpack backup.ab backup.tar ""
Распаковка архива осуществляется с помощью программы 7zip.

Итог: возможность открытия файлов из каталога приложения на компьютере, не имея на телефоне
прав Администратора (root)
